# MyNixOS

My NixOS setup

## Structure

The setup supports multiple machines (NixOS and macOS) with shared configuration where possible. The structure is as follows:

```text
~/.nixos/                              # rename to ~/dotfiles/ or ~/nix-config/
├── flake.nix
├── flake.lock
│
├── hosts/
│   ├── thinkpad/                      # NixOS machine
│   │   ├── default.nix
│   │   └── hardware-configuration.nix # generated by nixos-generate-config
│   ├── vbox/                          # NixOS machine
│   │   ├── default.nix
│   │   └── hardware-configuration.nix # generated by nixos-generate-configs
│   ├── mac1/                          # macOS machine (nix-darwin)
│   │   └── default.nix                # no hardware-configuration.nix needed
│   └── mac2/
│       └── default.nix
│
├── modules/
│   ├── nixos/                         # NixOS-only system modules
│   │   ├── common.nix
│   │   ├── nvidia.nix
│   │   ├── ...
│   │   └── virtualisation.nix
│   ├── darwin/                        # macOS-only system modules
│   │   ├── common.nix                 # brew, mac defaults, dock, etc.
│   │   └── developer.nix              # xcode tools, rosetta
│   └── shared/                        # platform-agnostic (rare at system level)
│       └── nix-settings.nix           # flakes, gc, substituters
│
└── home/                              # Home Manager — shared across ALL machines
    ├── default.nix
    ├── neovim.nix
    ├── shell.nix
    ├── git.nix
    └── packages.nix
```

## First-time setup

### NixOS (vbox)

```bash
# 1. Clone this repo
git clone https://github.com/aminfara/mynixos ~/dotfiles
cd ~/dotfiles

# 2. Replace the hardware-configuration placeholder
sudo nixos-generate-config --show-hardware-config > hosts/vbox/hardware-configuration.nix

# 3. Build and switch
sudo nixos-rebuild switch --flake .#vbox
```

### macOS (mac1)

```bash
# 1. Install Nix (Determinate Systems installer — includes flakes out of the box)
curl --proto '=https' --tlsv1.2 -sSf -L https://install.determinate.systems/nix | sh -s -- install

# 2. Clone this repo
git clone https://github.com/aminfara/mynixos ~/dotfiles
cd ~/dotfiles

# 3. Bootstrap nix-darwin (first time only — installs the `darwin-rebuild` binary)
nix run nix-darwin -- switch --flake .#mac1

# 4. Subsequent updates
darwin-rebuild switch --flake .#mac1
```

## Day-to-day

```bash
# Rebuild current NixOS machine
sudo nixos-rebuild switch --flake ~/dotfiles#vbox

# Rebuild current Mac
darwin-rebuild switch --flake ~/dotfiles#mac1

# Update all flake inputs (nixpkgs, home-manager, nix-darwin)
nix flake update

# Rollback the last switch
sudo nixos-rebuild switch --rollback   # NixOS
darwin-rebuild switch --rollback       # macOS
```

## Adding a new machine

1. `mkdir hosts/<name>`
2. Create `hosts/<name>/default.nix` importing the modules you need
3. NixOS only: copy the real `hardware-configuration.nix` into that folder
4. Add one line to `flake.nix` under `nixosConfigurations` or `darwinConfigurations`
5. Run the rebuild command with `--flake .#<name>`
